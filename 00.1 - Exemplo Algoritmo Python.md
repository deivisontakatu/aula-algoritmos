# 🧠 Algoritmo Completo em Python (Comentado)

Este exemplo reúne os principais conceitos de algoritmos e lógica de programação, incluindo variáveis, operadores, estruturas de controle, funções, vetores, matrizes e testes.

---

## 💻 Código em Python

```python
# ============================================
# 🧠 Exemplo Completo de Algoritmo em Python
# ============================================

# --------------------------------------------
# 📌 1. Tipos de dados e variáveis
# --------------------------------------------

# Variáveis armazenam valores na memória
numero_inteiro = 10          # tipo inteiro (int)
numero_real = 5.5            # tipo real (float)
texto = "Exemplo"            # tipo string (str)
valor_logico = True          # tipo booleano (bool)

# Exibição de valores
print("Valores iniciais:")
print(numero_inteiro, numero_real, texto, valor_logico)

# --------------------------------------------
# 📌 2. Operadores
# --------------------------------------------

# Operadores aritméticos
soma = numero_inteiro + 5
subtracao = numero_inteiro - 2
multiplicacao = numero_inteiro * 2
divisao = numero_inteiro / 2

# Operadores relacionais e lógicos
# Verifica se o número é maior que 5 E o valor lógico é verdadeiro
if numero_inteiro > 5 and valor_logico:
    print("Condição lógica verdadeira")

# --------------------------------------------
# 📌 3. Estrutura condicional
# --------------------------------------------

# Estrutura de decisão (if, elif, else)
if numero_inteiro < 5:
    print("Número menor que 5")
elif numero_inteiro == 10:
    print("Número igual a 10")
else:
    print("Número maior que 5 e diferente de 10")

# --------------------------------------------
# 📌 4. Estruturas de repetição
# --------------------------------------------

# Loop FOR: executa um número definido de vezes
print("\nLoop FOR:")
for i in range(1, 6):
    print("Valor de i:", i)

# Loop WHILE: executa enquanto a condição for verdadeira
print("\nLoop WHILE:")
contador = 1
while contador <= 3:
    print("Contador:", contador)
    contador += 1  # incremento

# --------------------------------------------
# 📌 5. Funções (sub-rotinas)
# --------------------------------------------

# Função que calcula a média de três valores
def calcular_media(a, b, c):
    # Retorna o resultado da média
    return (a + b + c) / 3

# Chamando a função
resultado_media = calcular_media(7, 8, 9)
print("\nResultado da média:", resultado_media)

# --------------------------------------------
# 📌 6. Vetores (listas)
# --------------------------------------------

# Vetor é uma lista de elementos do mesmo tipo
lista_numeros = [1, 2, 3, 4, 5]

print("\nElementos do vetor:")
for numero in lista_numeros:
    print(numero)

# --------------------------------------------
# 📌 7. Matrizes (listas de listas)
# --------------------------------------------

# Matriz é uma estrutura bidimensional
matriz = [
    [1, 2],
    [3, 4]
]

print("\nElementos da matriz:")
for linha in matriz:
    for elemento in linha:
        print(elemento, end=" ")
    print()

# --------------------------------------------
# 📌 8. Teste simples (teste unitário)
# --------------------------------------------

# Função de teste para validar a média
def teste_media():
    # Verifica se a função retorna o valor esperado
    assert calcular_media(10, 10, 10) == 10
    print("\nTeste da função passou!")

# Executando o teste
teste_media()

# --------------------------------------------
# 📌 9. Entrada de dados e uso completo
# --------------------------------------------

print("\n--- Sistema de exemplo ---")

# Entrada de dados do usuário
valor = int(input("Digite um número: "))

# Estrutura condicional + operador
if valor % 2 == 0:
    print("Número par")
else:
    print("Número ímpar")

# Uso de repetição (tabuada)
print("\nTabuada:")
for i in range(1, 11):
    print(valor, "x", i, "=", valor * i)
```
